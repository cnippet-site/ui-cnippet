## Quick Installation Guide

<Steps>
<Step>Create a Next.js Project</Step>

Create a new Next.js project with TypeScript and Tailwind CSS:

```bash
npx create-next-app@latest
```

<Step>Download Components</Step>

You can get all the components from our GitHub repository:

```bash
git clone https://github.com/cnippet-site/all-elements
```

Or download them directly from our [GitHub repository](https://github.com/cnippet-site/all-elements/). The repository includes all components, examples, and additional utilities.

<Step>Add the utility helper</Step>

Create a `lib/utils.ts` file to manage Tailwind CSS classes. This utility helps combine Tailwind classes efficiently and handles class name conflicts:

```bash
npm install clsx tailwind-merge
```

```tsx
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
}
```
